Análise de Padrões e Tendência da Indústria de Jogos

Este projeto de Processamento de Linguagem Natural (NLP) e Machine Learning teve como objetivo analisar e identificar padrões nos resumos de jogos da plataforma Metacritic. Utilizando técnicas como TF-IDF para vetorização de texto e o algoritmo K-means para clusterização, o estudo agrupou jogos com base em semelhanças linguísticas, revelando uma forte correlação entre a linguagem dos resumos e os gêneros dos jogos. A análise final demonstrou como descrições estrategicamente elaboradas podem impactar positivamente a percepção crítica e o desempenho comercial dos títulos.
